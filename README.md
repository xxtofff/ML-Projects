# Machine Learning and Data Science Projects

This repository contains a collection of machine learning and data science projects aimed at developing practical modeling skills across multiple domains, including deep learning and time series analysis. Each project follows an end-to-end workflow covering data preprocessing, feature engineering, model development, and evaluation.

## Projects

### 1. Customer Churn Prediction
A deep learning–based model for predicting customer churn using the IBM Telco dataset. The pipeline includes data preprocessing (categorical encoding and feature scaling), train-test splitting, and training a feedforward neural network implemented in Keras/TensorFlow using binary cross-entropy loss and the Adam optimizer. Model performance is evaluated using accuracy and a confusion matrix.

### 2. Inflation Forecasting *(Under Construction)*
A time series analysis project focused on forecasting inflation using BSP data. Current work includes data preprocessing, stationarity testing, and exploratory analysis. Planned methods include ARIMA/SARIMA and related forecasting techniques, with ongoing evaluation and model refinement.
